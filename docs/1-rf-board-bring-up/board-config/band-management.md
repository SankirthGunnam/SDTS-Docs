## Overview

Band Management controls which RF bands are enabled and how they are configured for the board. Use this section to add, remove, or modify band entries.

## Supported Bands

Configure the list of bands that the device supports. Each band entry includes frequency range and related parameters. You can import a preset or define bands manually.

## Adding a Band

To add a new band, specify the band name, start and stop frequencies, and any band-specific options. The configuration is validated before being applied.

## Removing a Band

Bands can be removed from the active list. Ensure no dependent configurations reference the band before removal.

## Band Presets

Presets provide quick setup for common band combinations. Select a preset to apply a predefined set of bands, then adjust as needed.

## Validation and Apply

After changing band settings, run validation to check for conflicts. Use **Apply** to write the configuration to the board.
