# SDTS Introduction

This document describes the **SDTS** (Sample Documentation Tool Suite) and how it fits into your workflow.

## What is SDTS?

SDTS is a documentation tool that helps you:

- Convert Confluence content to structured documentation
- Publish documentation as HTML with clear navigation
- Maintain a single source of truth in Markdown

## Key Features

- **Left navigation**: Browse all documentation pages from the sidebar
- **Right navigation**: Jump to sections on the current page
- **Reusable template**: All pages use the same Jinja layout for consistency
